### Usage
Clone repository:
```bash
git clone https://github.com/m0zgen/update-openvas
```

Cd to repo-folder and make executable:
```bash
cd update-openvas && chmod +x update-openvas.sh
```

Execute script:
```bash
./update-openvas.sh
```

### Schedule
Copy script to /etc/cron.daily/:
```bash
cp update-openvas.sh /etc/cron.daily/
```
Enjoy!
